Familia adotada

O Design System usa Material Icons (Google) no estilo filled como familia padrao para todos os icones de interface. Icones sociais/brand usam os SVGs oficiais de cada marca.

Formato: SVG inline com viewBox="0 0 24 24" e fill="currentColor".

Tamanhos: size-6 (24px padrao) · size-5 (20px) · size-4 (16px trailing).

Cor: herdada do container via text-{cor} + fill="currentColor".

Partial: _partials/icon-button.php centraliza todos os paths. Novos icones devem ser adicionados la.
