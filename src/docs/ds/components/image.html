✓ Use quando

  • Sempre incluir overflow-hidden no wrapper para conter object-cover
  • bg-neutral-100 como cor de placeholder durante carregamento
  • rounded-sm (4px) em todos os wrappers de imagem

✗ Não use quando

  • Não aplicar comportamento hover/state — estados pertencem ao card pai
  • Sempre usar object-cover com w-full h-full
  • Não usar valores de aspect ratio fora dos 9 definidos
