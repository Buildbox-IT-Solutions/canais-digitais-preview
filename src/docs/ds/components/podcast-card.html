Podcast Card

Card de episódio de podcast com play overlay, meta linha duração+autor e suporte a imagem 16:9 e 1:1.

PropTipoPadrãoDescrição
size string medium 'large' | 'medium' | 'small' | 'xsmall'
orientation string vertical 'vertical' ou 'horizontal'
ratio string 16:9 '16:9' (aspect-video) ou '1:1' (aspect-square)

✓ Use quando

  • Mostrar meta combinada: "99h 99m · Por Author Name" ao invés de byline simples
  • aria-hidden="true" no separador bullet
  • Ratio 1:1 para listagens estilo capa de álbum

✗ Não use quando

  • Podcast Large V usa headline-sm (24px), não headline-md (28px) como News Card
  • Não usar byline simples "Por Author" — sempre incluir duração na meta
  • Não usar <button> para play overlay quando dentro de <a>
