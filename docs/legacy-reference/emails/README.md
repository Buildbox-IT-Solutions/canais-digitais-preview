# Templates de e-mail (referência legada)

Estes 4 arquivos HTML são templates de e-mails transacionais do protótipo PHP legado (`legacy/src/emails/`), copiados aqui verbatim antes da limpeza do diretório `legacy/`. Eles cobrem os fluxos de boas-vindas, confirmação de cadastro, exclusão de conta e recuperação de senha descritos em `legacy/FEATURE-cadastro.md`.

O app React atual não possui nenhuma tela de preview de e-mail nem conteúdo de e-mail mockado em lugar algum do repositório — por isso esta cópia é mantida apenas como referência, para o caso de os designs de e-mail entrarem em escopo futuramente.

Estes arquivos **não estão conectados a nenhuma rota ou tela atual** do projeto; são material de consulta isolado.
